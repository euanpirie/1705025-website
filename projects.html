Arduino Group Project

For one of the modules, we were split into groups and tasked to make anything using an Arduino. My group came up with a game called Flash attack, where you need to press a button as the lights change to score a point. We used a switch to allow the user to change the speed the lights change. We linked the arduino to Processing, which allowed us to displays the users score, as well as allowing the user to change the volume of the buzzer that goes off when the user scores a point.
